# sfotdisasm: NES disassembler (6502)
Disassembler for 6502 NES/Dendy programs (iNES format).

## How to install
```bash
$ cd sfotdisasm && make && sudo make install
```

## How to use
```bash
$ sfotdisasm inputfile.nes [outputfile.asm]
```
